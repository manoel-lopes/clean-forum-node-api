# clean-forum-node-api

## 🎯 Business Rules

- [x] It must be possible to create a user
- [x] It must be possible to create a instructor
- [x] It must be possible to create a question
- [x] It must be possible to answer a question
- [x] It must be possible to choose the best answer for a question
- [x] It must be possible to delete a question
- [x] It must be possible to delete a answer
- [x] It must be possible to edit a question
- [x] It must be possible to edit a answer
- [] It must be possible to comment on an answer
- [x] It must be possible to get a question by the slug
- [x] It must be possible to list the most recent questions
- [x] It must be possible to list the answers of an question
- [] It must be possible to list the comments of an answer
- [] It must be possible to delete a comment on an answer

## ⚙️ Functional Requirements

- [x] ✅ It must be possible to authenticate a user;


## 🔒 Non-Functional Requirements

- [x] 🔑 A user must be identified by a JWT (JSON Web Token);
